版本对应支持

| MyBatis-Spring | MyBatis | Spring Framework | Spring Batch |   Java   |
|:--------------:|:-------:|:----------------:|:------------:|:--------:|
|      3.0       |  3.5+   |       6.0+       |     5.0+     | Java 17+ |
|      2.1       |  3.5+   |       5.x        |     4.x      | Java 8+  |


导入依赖：
```xml
<dependency>
    <groupId>org.mybatis</groupId>
    <artifactId>mybatis-spring</artifactId>
    <version>3.0.2</version>
</dependency>
```
